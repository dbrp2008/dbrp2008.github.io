How to use - Income Tracker

  • Add income sources as rows; add sub-sources with +Sub on any row.
  • Each cell can use its own currency - select it from the dropdown next to the amount.
  • Per-month currencies are independent: changing one month's currency doesn't affect others.
  • In Forecast mode, "Use 3-month avg" converts all historical values to USD before averaging, so mixed currencies average correctly.
  • Totals auto-link to the Expense Tracker income panel and the Tax Calculator.
  • Undo/Redo via Ctrl+Z / Ctrl+Y or the ↩↪ toolbar buttons.
